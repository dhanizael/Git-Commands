# Git Commands

> ***git config***

---------------------------------------

<strong> Display information about previous commits </strong>

```github
git log

```
----------------------------------------

> ***git config***

---------------------------------------

<strong> Display information about previous commits (detailed) </strong>

```github
git log --summary

```
----------------------------------------

> ***git config***

---------------------------------------

<strong> Display information about previous commits (briefly) </strong>

```github
git log --oneline	

```
----------------------------------------
